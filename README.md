# Project Rio ASM/C

A repo for all of the gecko codes made by Project Rio. Utilizes CGecko to maximize convenience and efficiency for modders.

## Setup
1. Clone this repo using

```bash
git clone --recurse-submodules https://github.com/ProjectRio/ProjectRio-ASM
```

2. Pull submodules if not present already using

```bash
git submodule update --init --recursive
```

3. Copy/Paste `config.template.json` from the CGecko folder to your root directory. Change filename to `config.json` and include your configurations.

## Tips
- From here, the rest of the setup will be learning how to use [CGecko](https://github.com/LittleCoaks/CGecko?utm_source=chatgpt.com). Please see the README/Wiki in that repo before beginning your mods.
- Check out existing codes here in the Gecko Codes folder to see how other mods are typically written
- Update the submodules every so often to make sure you're on the latest version of CGecko

### Happy modding!
