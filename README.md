# Project Rio ASM/C

A repo for all of the gecko codes made by Project Rio. Utilized CGecko to maximize convenience and efficiency for modders.

---

## Setup
1. Clone this repo using `git clone --recurse-submodules <https://github.com/ProjectRio/ProjectRio-ASM>`
2. Pull submodules if not present already using `git submodule update --init --recursive`
3. Copy/Paste `config.template.json` from the CGecko folder to your root directory. Change filename to `config.json` and include your configurations


From here, the rest of the setup will be learning how to use [CGecko](https://github.com/LittleCoaks/CGecko). Please see the README/Wiki in that repo before beginning your mods.

Happy modding!
